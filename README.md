# RiskMap_AI 🗺️

## IA de prédiction des zones à risque urbain et système d'alerte géolocalisé

### Description
Ce projet utilise le Machine Learning pour prédire les zones à risque criminel
dans une ville, à partir du Chicago Crime Dataset (2024–2026).

Il produit une carte interactive colorée (vert/orange/rouge) ainsi qu'un système
d'alertes automatiques.

### Technologies utilisées

- Python 3.13
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Folium
- Streamlit
- Plotly
